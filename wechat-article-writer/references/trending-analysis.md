# Trending Analysis

> Automated trending topic fetch and viral article structure analysis

## Usage

### Method 1: Direct Conversation
Tell me:
- "帮我找一下AI工具相关的热点"
- "分析一下最近设计领域的爆款文章"
- "搜索MidJourney相关的热门话题"

### Method 2: Structured Analysis
```
/热点分析 [领域/关键词]
```

## Analysis Dimensions

### 1. Trending Topic Extraction
- Latest tool releases/updates
- Industry trending events
- Popular trend topics
- User discussion hotspots

### 2. Viral Title Analysis
For found viral articles:
- Title type (Contrast/Question/Number/Negation)
- Character count
- Keyword extraction
- Emotional trigger points

### 3. Article Structure拆解
Analyze viral articles:
- Opening hook style
- Paragraph structure
- Image strategy
- Interaction design

### 4. Borrowable Angle Suggestions
Based on trend analysis:
-切入 angles
- Differentiated views
- Suitable topic directions for user

### Relevance
